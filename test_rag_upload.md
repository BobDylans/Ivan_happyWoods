# RAG 测试文档

## 什么是 RAG？

RAG (Retrieval-Augmented Generation) 是一种结合了检索和生成的 AI 技术。

### 核心组件

1. **检索系统**: 从知识库中查找相关信息
2. **向量数据库**: 存储文档的向量表示
3. **生成模型**: 基于检索到的信息生成回答

### 优势

- 提供更准确的答案
- 可以引用具体来源
- 支持动态更新知识库
- 减少模型幻觉

### 应用场景

RAG 特别适合以下场景：

- 企业知识库问答
- 客户服务机器人
- 技术文档助手
- 个人笔记管理

### 工作流程

```
用户提问 → 向量化 → 检索相关文档 → 注入上下文 → LLM 生成回答
```

这样可以确保 AI 的回答基于真实的知识库内容。
